# Address book

Address book using React, Redux, Node, MongoDB, Apollo Client / Server and GraphQL

## Getting Started

### Installing

```
git clone https://github.com/jayjongcheolpark/address-book.git
cd address-book

#for backend
cd backend
npm install
npm start

#for frontend
cd frontend
npm install
npm start

```
